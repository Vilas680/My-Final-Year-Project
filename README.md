# тЪЩя╕П Installation Guide

## Step 1: Clone Repository

```bash
git clone https://github.com/your-username/My-Final-Year-Project.git
```

## Step 2: Move Project to XAMPP

Copy project folder into:

```bash
C:/xampp/htdocs/
```

## Step 3: Import Database

Open **phpMyAdmin**

Import:

* users.sql
* admin.sql
* login_logs.sql
* forgot_logs.sql
* help_requests.sql

## Step 4: Run Project

Start:

* Apache
* MySQL

Open Browser:

